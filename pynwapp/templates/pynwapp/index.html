We are a group that meets monthly for talks, workshops and other fun stuff centered around the Python programming language.

We welcome pythonistas of all ages and experience levels, so whether you're a professional developer, hobbyist, learner or simply curious about coding in general, we aim to run sessions for everyone.

To find out more about our next meet-up, check out our group on meetup.com where you can sign up to be notified, or alternatively you can subscribe directly to the calendar feed in your favourite calendar app.

We're also on Twitter as @pythonnw and we have a mailing list on google groups.

What is Python?

Python is the name of a computer programming language created by Guido van Rossum in 1991. Today, Python's friendly syntax and pragmatic "batteries-included" standard library make it a popular choice for web applications, data science, and in the classroom.

To learn more, see python.org or check out an online tutorial at codecademy.com.

Talks and Workshops

We host talks on a wide variety of Python-related topics. If there is a subject in particular you would like to hear about, or if you would like to present a talk at one of our sessions, please contact us via Twitter.

We also run workshops from time to time, where folks can come along and get hands-on with some Python code. We recommend bringing along a laptop if you have one, and installing Python beforehand to get up and running smoothly. But if not, don't worry as there will be somebody there to pair up with and/or lend a hand.

Where to Find Us

Our group is hosted by the splendid co-working and community event space The Federation, which is located at Federation House, 2 Federation Street, Manchester, M4 4BF (close to Victoria station and Shudehill tram stop).

For more info about the space and the other events that take place there, check out www.thefederation.coop.

Source Code

We have a group on GitHub where we share code for workshops and community projects, including this website! Fixes and enhancements are welcome; feel free to send us a pull request.
